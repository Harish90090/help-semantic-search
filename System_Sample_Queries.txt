========================================
HOW TO USE THE FILTERS
========================================

1. Select SYSTEM in the sidebar (Patheon / TunnelWatch / SiteWatch)
2. Select FILE TYPE to narrow by media:
      Text        = pure text articles only
      Image+Text  = articles with screenshots
      Audio       = spoken audio guides
      Video       = video walkthroughs
3. Type a query and press Search
